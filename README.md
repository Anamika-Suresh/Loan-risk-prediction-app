# **PROJECT :  Loan Default Risk Prediction & Credit Strategy Optimization**

## Dataset Used :

The dataset used for this project was taken from the kaggle repository : https://www.kaggle.com/datasets/tahmidmir/credit-risk-dataset/code

## Objective :

Build an end-to-end ML system to predict loan default and explain which factors drive risk, enabling better lending decisions.

##  Input Features
- Investment Total
- Current Balance
- Marital Status
- Gender
- Install Size
- Due Payment
- Compensation Charged
- Client Type
- Repay Mode

# Phase 1: EDA and Data Pre-processing

**Preprocess and Cleaning**

- Checking for any irrelevant features in the dataset and dropping them.

- Checking for null / missing values.

- Check for different data types.

**Exploratory Data Analysis through Visualisation**

- Boxplot (Handle outliers)

- Heatmap

# Phase 2: Feature Selection and Model Building

**Models Used**

- Logistic Regression
  
- Random Forest
   
- XGBoost
